Spaceship Mission — это 2D игра в жанре point-and-click квест, разработанная с использованием Godot Engine под операционную систему Аврора в качестве учебного проекта. 

### Сюжет

На космической станции произошла авария. Вы оказались взаперти в одном из инженерных модулей. Ваша миссия — выбраться оттуда и спасти экипаж. По ходу квеста вам необходимо разобраться в поломке и устранить ее, решая различные логические задачи. Когда будет пройдено последнее задание, неисправность на корабле будет разрешена и игра завершится успешным прохождением миссии.

### Особенности игры

- Космическая тематика: сюжет разворачивается на космическом корабле, который надо починить;
- Квест построен на прохождении головоломок разного уровня сложности;
- Оформление в sci-fi стиле;
- В игре присутствуют анимации, созданные с помощью Godot Engine и способствующие визуальному разнообразию.

### Технологический стек

- Движок: Godot Engine 4
- Языки программирования: GDScript, C#
- Графика: 2D, растровая пиксель-арт
- Аудио: Ogg Vorbis
- Операционная система: Аврора
- Система контроля версий: GitHub

### Процесс игры

1. В начале квеста игрок понимает, что персонаж застрял на космическом корабле, и ему необходимо оттуда выбраться.
2. Перемещаться между локациями корабля можно с помощью кликов левой кнопкой мыши.
3. Есть возможность пополнять инвентарь вещами, необходимыми для прохождения квеста, а также пользоваться подсказками при затруднении.
4. Можно открыть меню, в котором будет предоставлен выбор: выйти из игры, поставить на паузу или повторно ознакомиться с сюжетом.
5. За решение головоломок открываются новые зоны корабля, и вы получаете возможность продвигаться дальше по сюжету.
6. После выполнения всех заданий поломка считается устраненной, а квест — пройденным
